Was macht PhantomJS

PhantomJS = Headless WebKit skriptbar durch JavaScript (macht Verbindung zu behat möglich)

Was macht CasperJS

Casper = Definiert und ordnet Navigation Schritte & macht Screenshots

Was macht Resemble.js

Resemble.js = vergleicht Screenshots auf RGB-Pixel-Basis (benutzt HTML5 File API um Bilder-Daten zu parsen)

Was macht PhantomCSS

PhantomCSS = Generiert die Diffs der einzelnen Screenshots
